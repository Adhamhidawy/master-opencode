         }

    3. Create `src/components/interactive/search-overlay.tsx` as a `"use client"` component:
       - Import `useState`, `useEffect`, `useRef`, `useCallback` from React
       - Import `useRouter` from `next/navigation`
       - Import `search` and `SearchResult` from `@/lib/search`
       - Import `cn` from `@/lib/utils`
       - Import `Search`, `X` from `lucide-react`

       - State: `isOpen: boolean`, `query: string`, `results: SearchResult[]`, `selectedIndex: number`
       - Input ref for auto-focus

       - Keyboard event listener (useEffect on document):
         - `(e.ctrlKey || e.metaKey) && e.key === "k"` → `e.preventDefault()`, toggle `isOpen`
         - `e.key === "Escape"` → close overlay
         - When overlay is open:
           - `e.key === "ArrowDown"` → increment selectedIndex (wrap)
           - `e.key === "ArrowUp"` → decrement selectedIndex (wrap)
           - `e.key === "Enter"` → navigate to `results[selectedIndex].href`, close overlay

       - When `isOpen` changes to true: clear query, focus input
       - When `query` changes: call `search(query)`, update `results`, reset `selectedIndex` to 0

       - Render structure (only when `isOpen` is true):
         - **Backdrop**: `fixed inset-0 z-[60] bg-black/50 backdrop-blur-sm` — onClick closes overlay
         - **Overlay panel**: `fixed top-[15%] left-1/2 -translate-x-1/2 z-[70] w-[90vw] max-w-[600px]`
           - **Input area**: `bg-bg-2 border border-border rounded-t-lg p-4 flex items-center gap-3`
             - Search icon (muted color)
             - Input: `bg-transparent outline-none text-text placeholder:text-text-3 flex-1 text-[.95rem]` with placeholder "Search lessons, commands, challenges..."
             - Close button (X icon): `text-text-3 hover:text-text` — closes overlay
           - **Results area**: `bg-bg-2 border-x border-b border-border rounded-b-lg max-h-[400px] overflow-y-auto`
             - If query empty: show "Type to search across all content..." in muted text
             - If no results: show "No results found" in muted text
             - If results: map over them rendering result items:
               - Each item: `px-4 py-3 cursor-pointer flex items-center gap-3 transition-all`
               - Selected state: `bg-bg-3` (when index === selectedIndex)
               - Type badge: small colored tag showing type
                 - "Lesson": `bg-accent/10 text-accent-2 text-[.7rem] font-bold uppercase px-2 py-0.5 rounded`
                 - "Cheat Sheet": `bg-blue/10 text-blue text-[.7rem] font-bold uppercase px-2 py-0.5 rounded`
                 - "Challenge": `bg-orange/10 text-orange text-[.7rem] font-bold uppercase px-2 py-0.5 rounded`
               - Title: `text-text text-[.88rem] font-medium`
               - Description: `text-text-3 text-[.78rem] truncate`
           - **Footer hint**: `px-4 py-2 border-t border-border bg-bg-3 text-[.72rem] text-text-3 flex gap-4`
             - "↑↓ Navigate" · "Enter Select" · "Esc Close"
